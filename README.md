# Sipster üç∏

**Sipster** is a modern cocktail platform built with **TypeScript**, **Next.js**, **Prisma**, **PostgreSQL**, and **Shadcn UI**. It allows users to explore, discover, and manage their favorite cocktails in a sleek, user-friendly interface.

---

## Features

- **Browse Cocktails with Infinite Scroll:** Explore a curated list of approximately 100 popular cocktails, loaded dynamically as you scroll.
- **Add to Favorites:** Save your favorite cocktails for quick access later.
- **Responsive Design:** Fully responsive UI built with Shadcn UI components. Dark and Light Mode.
- **User Authentication:** Secure sign-up and login functionality.
- **Type-safe Backend:** Built with Prisma and PostgreSQL for reliable and scalable data management.
- **Fast and Interactive:** Modern Next.js features ensure smooth navigation and interactions.
---

## Tech Stack

- **Frontend:** Next.js, TypeScript, Shadcn UI, Tailwind CSS
- **Backend:** Prisma, PostgreSQL, Next.js API Routes
- **Authentication:** NextAuth
- **State Management:** React Hooks (useState, useEffect, Context API)
